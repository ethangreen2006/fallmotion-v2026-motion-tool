## What is FallMotion?

FallMotion is a browser-first motion graphics editor centered on SVG scenes and a keyframe timeline. It keeps scene building, timing, and previewing in the same workflow so you can shape animated visuals without bouncing between tools.

The experience is built to stay lightweight and direct. It suits creators who want a practical way to sketch animation ideas, experiment with easing behavior, and reuse templates. Export options include WEBM output via MediaRecorder and an embedded GIF89a encoder for animated GIFs.

---

## Features

- SVG scene editing with a timeline built for keyframe-based animation
- Four easing options for shaping how motion accelerates and settles
- Four templates to speed up common animation setups
- WEBM export powered by MediaRecorder
- Inline GIF89a encoder for animated GIF output
- Web-first workflow with no desktop install required
- Autopilot mode support for streamlined animation handling
- Built for motion graphics work inspired by After Effects-style workflows

## How to use it

1. Open FallMotion in a web browser.
2. Create or load an SVG scene.
3. Place animation changes on the keyframe timeline.
4. Choose an easing preset that fits the motion style.
5. Apply a template when you want a faster starting point.
6. Preview the sequence and adjust timing as needed.
7. Export to WEBM or GIF89a depending on the format you need.

For best results, keep scene elements organized before exporting so timeline edits stay manageable.
